# パスワードマネージャー

ターミナル上で使用できるパスワード管理ツールです。
サービス名、ユーザー名、パスワードをセットで登録し、すばやく検索・取得できます。

## 主な機能

- パスワードの登録(Add Password)：サービス名、ユーザー名、パスワードをセットで登録します。
- パスワードの取得(Get Password)：登録済みのサービス名からユーザー名とパスワードを取得します。

## 使い方

1. シェルスクリプトの実行

    ```bash
    ./password_manager.sh
    ```

1. メニューの選択

    起動すると以下のメニューが表示されます。

    ```bash
    パスワードマネージャーへようこそ！
    次の選択肢から入力してください(Add Password/Get Password/Exit)：
    ```

    - パスワードを登録したい場合

        `Add Password`または`a`を入力し、画面の指示に従って「サービス名」「ユーザー名」「パスワード」を入力してください。

    - パスワードを確認したい場合

        `Get Password`または`g`を入力し、画面の指示に従って「サービス名」を入力してください。

    - 終了したい場合

        `Exit`または`e`を入力してください。

## 注意事項

- 先頭・末尾の空白等について

    入力時、先頭および末尾にあるスペースやタブは削除されます。文字列の中間にある場合は保持されます。

- 使用できない文字

    データ管理の仕組み上、コロン(`:`)は使用できません。
